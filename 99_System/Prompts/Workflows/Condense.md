# Workflow: Condense

## Objective

現在の長い会話を、次の応答または別 LLM へそのまま渡せる短い「会話コンテキスト要約」に圧縮する。

## Scope

- 対象: 現在の会話コンテキスト（Vault ファイルは書き換えない）
- 原則としてファイル保存は行わない。長期保存が必要な内容は `crystallize` に委ねる。
- 別 LLM への引き継ぎにも使えるよう、Vault 外の受け手を想定した粒度で書く。

## Procedure

1. 会話全体を走査し、目的・現在状態・決定事項・制約・未解決事項・次アクションを抽出する。
2. 検討過程、却下済み候補、冗談、検索結果全文など、次の作業に不要な情報を「持ち越さない情報」に分類する。
3. 固定出力形式に沿って要約を生成する。
4. ユーザーに提示する（ファイル保存はしない）。

## Output Format

```md
# Condensed Context

## 目的

## 現在状態

## 決定事項

## 有効な制約

## 未解決事項

## 次アクション

## 参照

## 持ち越さない情報
```

## Rules

- セクション順は固定。省略不可。内容がなければ「なし」と明記する。
- 後続エージェントが即座に作業再開できる粒度に絞る。
- 長期保存すべき知識・決定が出た場合は、`condense` 後に `crystallize` の実行を提案する。
