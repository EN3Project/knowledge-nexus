# Memory Archive

`99_System/Memory/Archive/` は、現在の `run / load` や `INDEX.md` からは不要になったが、履歴として残す価値がある Memory を移す場所。

## Rules

- Memory を Archive に移動するのは `prune-memory / 記憶整理` の対象作業。
- 移動は必ずユーザー確認後に行う。
- Archive へ移した Memory は原則として `run / load` の復元対象にしない。
- 完全削除より Archive 移動を優先する。
- Archive へ移した場合は、必要に応じて `99_System/Memory/INDEX.md` の `Reference Memories` から外すか、Archive 済みとして注記する。
