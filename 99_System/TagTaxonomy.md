# Tag Taxonomy

ノート作成・抽出時はここに定義されたタグから選択する。
新タグが必要な場合はこのファイルを更新してから使用する。

**形式:** `kebab-case`、最大2階層（例: `ai/llm`）

---

## プロジェクト
- `project/active` — 進行中のプロジェクト
- `project/completed` — 完了したプロジェクト
- `project/on-hold` — 保留中

## 開発
- `dev/mcp` — Model Context Protocol
- `dev/api` — API 設計・実装
- `dev/frontend` — フロントエンド
- `dev/backend` — バックエンド
- `dev/infra` — インフラ・DevOps

## AI / LLM
- `ai/llm` — 大規模言語モデル全般
- `ai/prompt` — プロンプトエンジニアリング
- `ai/agent` — AIエージェント
- `ai/rag` — RAG・検索拡張生成

## 知識管理
- `pkm/method` — 知識管理の方法論
- `pkm/obsidian` — Obsidian 固有の設定・プラグイン
- `pkm/workflow` — 運用ワークフロー

## リサーチ
- `research/tech` — 技術調査
- `research/market` — 市場調査
- `research/paper` — 論文・学術

## その他
- `permanent` — 普遍的な知見（Permanent Note）
- `clipping` — 外部記事・参考資料
- `draft` — 下書き・作業中

---

*カスタマイズ推奨: 自分の運用に合わせてタグを追加・削除してください。*
